# MØRK Portfolio Manager

Aplicación fullstack para gestionar proyectos de diseño web premium.
Desarrollada con React, TypeScript, Tailwind CSS y Express.

## 🚀 URLs de producción
- **Frontend**: [mork-portfolio.vercel.app](https://mork-portfolio.vercel.app)
- **API**: [mork-api.onrender.com](https://mork-api.onrender.com)
- **Trello**: [Enlace al tablero](PEGA AQUÍ TU ENLACE DE TRELLO)

## 🛠️ Tecnologías
- React + TypeScript + Tailwind CSS
- Node.js + Express
- Vite
- Vercel (frontend) + Render (backend)

## 📁 Estructura
- `src/` — Frontend React
- `server/` — Backend Express
- `docs/` — Documentación completa

## 🚀 Desarrollo local
```bash
# Backend
cd server && npm install && npm run dev

# Frontend (otra terminal)
npm install && npm run dev
```
