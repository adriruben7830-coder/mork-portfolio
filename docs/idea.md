# MØRK Portfolio Manager

## ¿Qué problema resuelve?
Como diseñador web freelance necesito una herramienta para gestionar
y mostrar mis proyectos de diseño web de forma profesional. MØRK
Portfolio Manager permite crear, editar y organizar proyectos con
una estética oscura y premium que refleja la identidad de mi agencia.

## Usuario objetivo
Diseñadores web freelance y agencias pequeñas que necesitan gestionar
su portfolio de proyectos de forma visual y profesional.

## Funcionalidades principales
- Ver lista de proyectos con imagen, título y categoría
- Crear nuevos proyectos con formulario
- Editar proyectos existentes
- Eliminar proyectos
- Filtrar proyectos por categoría
- Vista detalle de cada proyecto
- Diseño responsive y modo oscuro

## Funcionalidades opcionales
- Drag & drop para reordenar proyectos
- Animaciones de transición entre páginas
- Buscador de proyectos en tiempo real
- Estadísticas del portfolio

## Mejoras futuras
