# Organización del proyecto

## Herramienta utilizada
He utilizado Trello para organizar el desarrollo del proyecto
siguiendo la metodología Kanban.

## Columnas del tablero
- **Backlog**: ideas y tareas pendientes de priorizar
- **Todo**: tareas listas para empezar
- **In Progress**: tareas en las que estoy trabajando ahora
- **Review**: tareas terminadas pendientes de revisar
- **Done**: tareas completadas y revisadas

## Cómo organizo el trabajo
Cada funcionalidad del proyecto tiene una tarjeta en Trello.
Las tarjetas grandes las divido en subtareas técnicas más pequeñas.
Voy moviendo las tarjetas entre columnas según avanzo.
Intento no tener más de 2-3 tareas en progreso a la vez para
mantener el foco.
