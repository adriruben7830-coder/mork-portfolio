# Reflexión final

## ¿Qué aprendí?
Este proyecto me ha enseñado a conectar un frontend React con
un backend Express, algo que no había hecho antes. Entender
cómo fluyen los datos desde la interfaz hasta la API y viceversa
ha sido clave para comprender el desarrollo fullstack.

TypeScript me ha ayudado a detectar errores antes de ejecutar
el código, aunque al principio me costó entender las interfaces
y los tipos genéricos como Omit y Partial.

## Principales problemas encontrados
- Configurar TypeScript en el backend fue complicado al principio
- Los errores de importación de tipos me bloquearon un rato
- Conectar frontend y backend con CORS requirió investigación
- Entender el flujo Context → Hook → Componente llevó tiempo

## Cómo utilicé IA durante el desarrollo
- Claude me ayudó a estructurar la arquitectura del proyecto
- Usé IA para generar la base de los componentes y luego los adapté
- La IA me explicó conceptos como Context API y custom hooks
- Para la documentación, usé IA como punto de partida y luego
  revisé y personalicé el contenido

## Conclusión
El desarrollo fullstack es más complejo de lo que parece pero
muy satisfactorio cuando ves que todo se conecta. La clave
es ir paso a paso y no intentar hacer todo a la vez.
